branch. It is to let the coupled `HC-DU-011 + HC-DU-022` program construct the

influence reopener are the same staged research program viewed at different
resolutions.

## 2. Contested finding

### Exact current proposition

The synthesis and live lane state say, in substance:

> Reopen the influence branch only when one construction supplies all three:
> an independently motivated response law, a matched embedding of a held-out
> incomparable pair, and a unit-bearing map from the response to the proposed
> physical observable.

### Why it is too broadly scoped

Let:

- `R` = an independent response law;
- `E` = a matched physical embedding of the held-out pair; and
- `U` = a unit-bearing observable map.

The conjunction `R ∧ E ∧ U` is appropriate before saying that a functional has
been physically selected **and** mapped to a nonzero physical observable. It
does not follow that no scientifically meaningful result exists at a proper
subset:

| Available object | Legitimate result | Still forbidden |
|---|---|---|
| `R` | conditional functional prediction on an abstract profile domain | physical selector or Lambda identity |
| `R ∧ E` | held-out physical selector test | absolute scale or Lambda value |
| `U` plus a raw carrier | metrological scale test | functional selection |
| `R ∧ E ∧ U` | candidate physical identification | banked identity without robustness and rival tests |

The proposition is therefore a **promotion rule**, not a theorem of
epistemic indivisibility.

### Evidence that could overturn this criticism

The monolithic gate would be vindicated if it were shown that, for the actual
DU objects, defining any one of `R`, `E`, or `U` necessarily and
non-conventionally fixes the other two. No such dependence appears in the
current equations or receipts.

## 3. Bounded swing: a separability witness

I used the already frozen incomparable pair:

```text
pL = [37,1,1,1]/40
pR = [36,4,0,0]/40.
```

Consider a response model specified independently of the three concentration
names:

```text
Y(p) = sum_i p_i X_i,
E[X_i] = 0,
Cov(X_i,X_j) = v delta_ij.
```

It entails:

```text
Var[Y(p)] = v sum_i p_i^2.
```

For the frozen pair:

```text
sum_i pL_i^2 = 0.8575
sum_i pR_i^2 = 0.8200
response difference = 0.0375 v > 0.
```

Meanwhile:

```text
H(pL)    = 0.348780
H(pR)    = 0.325083
Gini(pL) = 0.675
Gini(pR) = 0.700.
```

Thus the response law orders `pL` above `pR`, agreeing with the quadratic
participation reading while Shannon and Gini place the more concentrated
reading on `pR`. This is a genuine discriminating prediction conditional on
exchangeable independent perturbations.

The witness does **not** establish that the DU modes or records are
exchangeable independent perturbations, that these abstract profiles have
been embedded in one physical state space, that `v` has a DU-native value, or
that the response is Lambda. It is not evidence for participation as the
actual selector.

It is sufficient for the narrower logical point: an independent response law
can select a functional form before a unit-bearing cosmological map exists.
The three missing objects are therefore separable research stages.

### Swing disposition

> **NARROWED.**

- Upheld: `R ∧ E ∧ U` is required for the proposed physical-identity result.
- Overturned: all three are not required before any component research can
  legitimately resume.
- Unchanged: the present wave contains neither `R`, `E`, nor `U`, so its
  `SELECTOR-OPEN` and scale verdicts stand.
- Scope: this is an analytic countermodel about inferential structure, not a
  new physical candidate or claim.

**Output.** Two label-invariant causal-growth histories realizing the held-out

**Why now.** This turns a conditional response theorem into a physical

**Output.** A preregistered sign prediction from each surviving functional and

**Why now.** `R ∧ E` earns a selector result independently of whether the

**Output.** A raw carrier with declared dimensions, a beta function or

calibration-removal controls.

**Falsifier / stop.** Kill the scale leg if the nonzero value is supplied by
`m~H0`, `N0`, a present-age window, a fitted conversion constant, or a unit
choice that changes the physical conclusion.

**Why now.** This is the `HC-DU-022` burden. Doing it after selector
discrimination avoids attaching units to an arbitrary functional.

### Action 6 — integrate, ablate, and compare rivals

**Output.** One complete `R ∧ E ∧ U` receipt, completion-sensitivity analysis,
GU ablation, and comparison with the strongest causal-set, everpresent-Lambda,
and nonlocal-memory rivals.

**Falsifier / stop.** No physical-identity claim if the result disappears
under a legitimate completion, if a null produces the same response with fewer
imports, or if the units map does not survive the generated-scale controls.

**Why now.** Only this stage can assess abductive preference or prepare a
claim for independent hardening.

## 8. Missed or underused outcomes

### Normalization erases provenance

The exact-duplication result is more than a scale null. The same normalized
profile can arise from independent accretion or from copies that add no
information. Therefore `p` is a quotient not only of common magnitude but, by
itself, of causal provenance. The next order-first build should carry ancestry,
dependence, or common-past structure beside the weight vector.

### Mobility is constitutive, not merely a nuisance

The Euclidean/affine disagreement should not only be filed as “completion
unselected.” In dissipative theories, mobility can be a constitutive law whose
choice is experimentally exposed by transient response. The disagreement
creates a discriminating intervention opportunity.

### `NOT_EVALUABLE` is a design receipt

The label names the minimum experimental apparatus still missing. It should be
stored as a typed tuple `(question, missing objects, cheapest resolving test)`,
not only as an endpoint verdict.

### The pivot can satisfy the reopener

The synthesis presents “stop influence” and “build `HC-DU-011 + HC-DU-022`”
sequentially. A better scientific use is compositional: the causal-growth law
may supply the physical embedding and response, while dimensional
transmutation supplies the units map. The existing influence objects then
serve as downstream diagnostics rather than as discarded candidates.

### Undefined endpoints are partial-observable information

The Bianconi profiles are observables on the open domain `D>0`, not state
variables defined on the full stationary closure. Their endpoint
undefinedness is not merely “bad persistence”; it specifies the domain and
warns against treating normalized modal shares as ontic quantities.
