"record-accretion → becoming → dark energy" thesis. Treat each as a pre-registered kill the
transducer must pass before anything is banked.

  TaF — the finality repo — reads as *B-theory / block-compatible*: "apparent passage should be

- **The D-FORK / Gödelian-regime gate (TI, `explorations/E045-d-fork-synthesis-…`).**
  Record-accretion is genuine *issuance/becoming* only if the source is in the **Gödelian
  (self-generating) regime**; in the finite-type-space regime the same `N`-roll is *fixed-source
  disclosure* and the DE would be a **bookkeeping artifact, not becoming**. Which regime it is
  is *proven non-computable in general*. DU must state which regime its `N=e^{4p}` roll lives in
  and defend it; `E156` gives an executable N-curvature *signature* (not a decision procedure).
- **The N-wire-crossing discipline (TI, `E046`).** TI's `N` is a novelty **rate**; DU's `N` is an
  accreted record **count** (`Λ~1/√N`). *Different objects sharing a letter* — the structure
  transfers, the identity must not. Keep them typed apart.
- **Record-change ≠ finality (continuity-ledger, `explorations/candidate-transduction-atlas-seeds.md`).**
  A real, maintained, updated public record can have *zero* boundary force: "record change alone
  does not imply access change, entitlement change, or finality." Type record-accretion and
  finality *separately* — not every increment of `N` is a crossing.
- **The observer-gradient falsifier (TaF, `audits/2026-06-10-post-session-reassessment.md`).**
  Any observer/record/access-centred DE story predicts an observer-centred cosmological gradient;
  observation excludes such gradients. A cheap guardrail to run on the transducer before banking.
- **Type-Distance / "count of source activity" trap (continuity-ledger R0 result).** If `N` is
  *definitionally* the source action re-labeled (d=0), the "accretion" is trivial. Pre-empt when
  banking the roll↔accretion identification.
