# The self-authoring gossip-about-gossip Dirac, built

The first trunk attempt — naive Dirac spectral flow on a blindly-grown causal set — **nulled cleanly**,
build and kill converging object-for-object (spectral-flow-source-synthesis-2026-07-21.md). It failed
on three axes, and three independent lines (the kill, the build, and Joe's packet identification)
named the *same* repair: a **self-authoring, gossip-about-gossip-grown causal set whose own spectral
record feeds back to grow itself, with the Dirac operator on the order complex rather than the
1-skeleton.** This is the constructive build of that object, graded honestly per axis.

**Result: PARTIAL.** The repair crosses **exactly one** of the three axes — the one the naive flow
failed most completely (self-encoding / becoming). It does *not* lift the additive wall, and does
*not* generate a transmuted scale. A clean, non-inflated PARTIAL that names precisely what is
repaired and what is not.

## The object (the three named repairs, built at once)

A growing **gossip-about-gossip causal set** — `S` sites, each event references its self-parent
(its site's previous head) and other-parent(s) (heads learned via sync); its causal past
`K(e) = {e} ∪ K(parents)` is transitively closed, so **each event carries a record of its parents'
records = records of records = self-encoding**. Onto it, the three repairs:

1. **Growth = gossip-about-gossip forwarding**, not a fixed blind CSG law. Which other-parent an
   event references is biased by the current Dirac spectrum (see repair 3), so the growth law is
   *authored by the structure's own readout*, not a fixed external stochastic rule.
2. **D on the ORDER / REFERENCE complex, not the 1-skeleton.** Each event's reference set
   (`{e} ∪ parents(e)`) spans a simplex — "reference-chains populate the ≥2-cells." The discrete
   Dirac `D = ∂ + ∂*` lives on `C_0 ⊕ C_1 ⊕ C_2`; `dim ker D = Σ bₖ` (Hodge). This is a genuine,
   bounded 2-complex, unlike the *full* poset order complex, which truncates to a spurious giant `b_2`
   and is generically contractible (`b_1 = 0`, inert to a harmonic-keyed feedback — verified and
   discarded during construction).
3. **The spectrum is RECORDED BACK.** Every few steps, the Dirac harmonic 1-support (the degree-1
   zero-modes) is computed and a **spectral element** `se` is minted whose parents are that support;
   its knowledge-set `K(se)` becomes the **admissibility predicate** the next step consults. The
   observer (D's spectrum) now sits *inside* the admissibility space it defines — the Lawvere-L1 loop.

Positive controls are built into every axis (the DU charter's rule): the **1-skeleton** of the same
causal set (repair 2's control), the **blind** non-self-authoring growth (repair 3's control), and
the raw **gap-vs-weight** scaling (repair 3 / axis 3's control), plus teeth (RP², T², a genuine
1-loop RG). Probe: `tests/du_self_authoring_dirac_probe.py`, exit 0, 25/25, ~5 s.

## Axis 1 — non-additive W1 (torsion / cohomology ring): **NULL** (wall lifted, not populated)

**Pre-registered measure:** does the order complex's integer `H_•` carry TORSION (`Z/n`) or a
nonzero cup-product ring — `Θ_type > 1`, off the free-abelian `⊕Z` wall? **Positive control:** the
1-skeleton must be free (`H_1 = Z^{b_1}`, `H^2 = 0`, cup rank 0) — the additive wall, by the
structure theorem.

**What crossed (the structural lift, genuine):** the reference complex carries `H_2 ≠ 0` in **every**
run (best run f-vector `[34,122,109]`, Betti `[1,10,30]`), whereas its own 1-skeleton has Betti
`[1,89,0]` — `H^2 = 0`. A nonzero `H_2` is an invariant a 1-complex **provably cannot** carry. So
moving `D` off the link graph and onto the reference complex genuinely **escapes the free-`H_1`
theorem barrier** that walled the naive flow. The control fires: all 24 one-skeletons are free with
cup rank 0.

**What did NOT cross (the crossing itself):** across 24 runs *and* richer variants (grandparent /
"gossip-about-gossip-about-gossip" reference sets, larger cliques, denser feedback), the homology is
**entirely free**: **0/24 runs show any torsion; 0/24 show a nonzero cup product** (max cup rank 0).
So `Θ_type` stays **1** — the count is still `⊕Z`, now spread across degrees 1 and 2 rather than
degree 1 alone, but still a direct sum of `Z`'s. **W1 stays additive.**

**Mechanism / obstruction (the honest content of the null):** the gossip reference complex is a
*union of small simplices glued along shared faces* — generically homologically **free**. Integer
torsion `Z/n` requires a 2-cell attached along `n×` a 1-cycle (a degree-≥2, non-orientable-type
attaching map); a single gossip event contributes reference triangles each attaching with degree
`±1`, so **the dynamics never produce the attaching multiplicity torsion needs.** The teeth confirm
this is a real null, not a blind detector: the SNF catches `H_1(RP²) = Z/2`, and the GF(2) cup
product catches `H^1(T²)^{∪2} = 2`. The wall is *available* on the order complex; **this construction
does not reach it.**

## Axis 2 — first-person self-encoding (becoming): **CROSS (first-person)**

**Pre-registered measure:** does the self-authoring loop realize a genuine Lawvere structure — L1

  genuine knowledge-set `K(a)` on **0.68** of steps in the self-authoring loop, and **0.00** in the

**The honest caveat (pre-registered, loud).** The feedback rule is **fixed and deterministic**, so a

(Note a deliberate honesty trap the probe passes: the blind control *also* mints "fresh" predicates

## Axis 3 — cutoff-independent scale (transmutation): **NULL**

**Pre-registered measure:** does the recursive feedback self-generate a scale with a β-function / a
fixed point invariant under sprinkling-density rescale? **Positive control:** a fixed-cutoff/blind
version tracks the density (the naive failure).

- **Positive control fires (naive failure reproduced exactly).** The raw Dirac gap **is** the
  imported cutoff: `gap ∝ w` (link weight `w = 1/a`), `gap/w = 0.62051` constant to machine
  precision — cutoff-**dependent**.
- **No running coupling.** The dimensionless coupling `ĝ = gap/radius` is `w`-invariant *by
  construction* (a ratio of two things `∝ w`), so it cannot **be** the cutoff — but it does **not
  run**: `ĝ ≈ 0.18` across the roll and across the density knob, and **essentially the same for
  self-authoring (`[0.171, 0.170, 0.149]`) and blind (`[0.205, 0.179, 0.212]`)**. The feedback adds
  **no** cutoff-invariance beyond the trivial dimensionless-ratio artifact (SA spread 0.065 vs blind
  0.070 — not a distinction). **No β-function, no density-forgetting fixed point.**
- **Teeth:** a genuine 1-loop RG `Λ = μ·exp(−1/(2 b₀ g²))` is cutoff-invariant via a *real*
  β-function — which the gossip feedback does **not** reproduce.

**Verdict:** the recursive self-authoring loop does not self-generate a scale. The transmutation the
trunk needs is absent from this construction.

## Honest standing

installs the **first-person self-encoding loop** (axis 2) that the naive flow completely lacked —
a genuine, controlled Lawvere-L1 structure with the diagonal firing and the blind control confirming

sharpest honest form the reframe permits. It does **not** lift the **additive wall** (axis 1: the
order complex is homologically free — the wall is available but the gossip dynamics do not populate
it) and does **not** generate a **transmuted scale** (axis 3: no β-function; the coupling does not
run). So the trunk gets a foothold on the *inside/observer* leg but **not** on the two legs that
would source the dark-energy VALUE (non-additive count + sourced coefficient).

This is a **PARTIAL**, graded per-axis, not inflated to BREATHES. Against the pre-registration:
BREATHES would require all three; PARTIAL names which cross (axis 2, first-person) and which do not
(axes 1, 3) — and it names *why* each fails, mechanism-level, which is the useful product. It remains
a **conjecture** that closing the loop crosses the D-FORK; this build shows the loop can be **closed**
(the self-encoding is real and controlled) but that closing it **does not** by itself buy the
non-additive count or the sourced scale. The next honest target the two nulls name: an attaching-map
